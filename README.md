# Nelution

Nelution is a platform for launching instant, professionally designed websites for small businesses — restaurants, salons, gyms, retail shops, clinics, contractors, and similar service/product businesses.

Business owners get a hosted site built from an industry-specific template, with a no-code dashboard to manage everything themselves: menus, product catalogs, service listings, staff, hours, and photos, all toggleable in real time. Nelution handles hosting, custom domain setup, SSL, backups, and scaling, so owners don't need developers or technical skills to keep their site up to date.

Built with Next.js (App Router) and Supabase for authentication and data.
